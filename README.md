# Locardiacse
>一个2D横板射击游戏
## 简易场景所需的内容
   * GameSystem  
   挂载系统脚本文件(GameSystemInstance)  
   * Player  
   1.Player(tag为Player),挂载PlayerController脚本  
   2.ShadowController(tag为ShadowController)，用于伪阴影检测  
   * Background  
   背景图  
   * MainUI  
   游戏中主UI  
   * Light  
   伪光照范围(挂载MyLight脚本)  
   * bulletInstances  
   用于作为子弹对象池父物体
